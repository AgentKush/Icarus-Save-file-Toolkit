# Data Catalog

GitLab: icarus/data-catalog (ID: 5, private)
Route: /catalog

## Data Sources
- catalog_summary.json: Category index with entry counts
- full_catalog.json: All entries across all categories
Regenerated from data.pak after each game patch

## Features
- Browse 76+ game data categories
- Search and filter entries
- Patch diff viewer (what changed between versions)
- Read-only Blueprint, no database writes
